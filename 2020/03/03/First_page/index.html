hello 2020 3.3

时隔一年多，我终于搭建了博客。今天是2020年三月三日。记录一下我的搭建过程。

po一张鼬神的图

（我真的是哭着搭完的。。）

1.首先从github上注册账号，并且新建项目，项目名称切记写成yourgithubname.github.io

2.mac系统从终端安装npm,nodejs

按照命令安装以上内容

3.初始化博客，在/user下创建blog目录，在blog目录下执行hexo init，之后输入hexo s(server)命令即可在本地看到博客页面样式

4.将博客部署到github上，点击我们创建的项目，点击New repository, 主要注意以下内容

​ 1).配置自己的SSH密钥(keys)

​ 2).$ssh-keygen -t rsa -C "your_email@example.com"用该命令创建密钥

​ 3).出现如下框中的提示，一直回车，最终会生成属于你的ssh rsa密钥，进入提示中相关文档，cat密钥，点击github中的setting，将密钥填充。

​ 4).$ ssh -T git@github.com检测设置是否成功

​ 5).在hexo文件下执行如下命令,一定要进入hexo中

​ 6).访问 yourgithubname.github.io即可访问博客

一定要注意，你讲博客设置在哪一个文件夹下，最后执行hexo时，一定要进入文件夹，执行5中的命令。不然是404
